\textbf{Entropy Rate of Markov Process}

The Morse code uses dots, dashes, and two types of spaces: letter and word
spaces. Spaces only occur as delimiters; that is, no space can follow another
space. What is the entropy rate achievable under these constraints?  Hint: take
a look at Shannon’s 1948 paper for a start, and Example 5.1.3 of C\&T, but use
different notation. Write it out as a Markov process whose states are [.], [-],
[L- space], [W-space]. Assume that all allowed outputs from any state are
equally likely.

\begin{solution}

    The state-diagram is shown below.
    \includegraphics[scale=0.5]{./ss.pdf}

    First, we need to calculate the stationary distributuon $\mu$ of states such
    that $\mu P = P$. I found $\mu=\left[1/6,1/6,1/3,1/3\right]$ for
    \texttt{WS,LS,.,-}. Rest is easy due to Markov Property. Rate is given by
    $\lim H(X_n|X_{n-1}) = H(X_2|X_1)$.

    My calculation found it to be 1.667 bits per symbol. See the script 
    \url{http://github.com/dilawar/courses/raw/master/TASHIP/InformationTheory2019/HW4/prob1.py}
    for details (requires Python3.6+).

    \begin{verbatim}
['WS', 'DOT', 'LS', 'DASH']
[[0.   0.5  0.   0.5 ]
 [0.25 0.25 0.25 0.25]
 [0.   0.5  0.   0.5 ]
 [0.25 0.25 0.25 0.25]]

Stationary distribution:
['WS', 'DOT', 'LS', 'DASH']
[0.16666667 0.33333333 0.16666667 0.33333333]

Rate is: 1.667 bits per symbol 
    \end{verbatim}

    This means that there are $2^{1.667n}$ distinct strings of length $n$
    compared to $2^{2n}$ possible strings of length $n$.

\end{solution}

\question 

\textbf{Large Deviation Theory} We roll a fair six-sided die $n$ times, where
$n$ is large. What is the probability that the number of 2s is at least double 
the number of 1s in the resulting string? Of strings which satisfy this condition, 
what proportion of values will be 1s?

We can split this calculation into the following parts:

\begin{parts}
    
    \part[3] What is the constraint equation for the set $E$ of types (or empirical distributions)
    $P_{x^n}$ which satisfy the required condition?
    
    \begin{solution}
        That the number of 2s are at least twice as many as number of 1s i.e.
        $\Pr(2) \ge 2\Pr(1)$
    \end{solution}

    \part[5] Let $Q$ be the uniform distribution. Show that the member $P^*$ of $E$
    which minimizes the relative entropy $D(P^*||Q)$ is the maximum entropy
    distribution subject to the constraints from (a).

    \begin{solution}

    $Q=\left(1/6,1/6,\ldots 1/6\right)$ and 
    $P^*=\left(a, 2a, c, d, e, f\right)$ where $3a+c+d+e+f=1$.

    \begin{dmath}
        D(P^*||Q) = p(x)\log \frac{p(x)}{q(x)} \\
          = a\ln(6a)+2a\ln(12a)+c\ln(6c)+\ldots+f\ln(6f) \\
          = a\ln(a)+2a\ln(2a)+\ldots+f\ln(f)+\ln(6)\left(a+2a+c+d+e+f\right) \\
          = H(a,2a,c,d,e,f)+\ln(6)
    \end{dmath}
    I.e., maximizing $D(P^*||Q)$ requires maximizing $H(P^*)$.
    \end{solution}

    \part[5] Find the distribution $P^*$. E.g. let the proportion of 1s be an
    unknown $x$. Use some intuition to set the rest of the distribution. 
    Then use calculus to minimize. You can find the answer analytically.

    \begin{solution}
        Recall that uniform distribution maximized entropy. We set
        $c,d,e,f=b$. Now $3a+4b=1\implies b= (1-3a)/4$. This gives us,

        \begin{align}
            H(P^*) &= H(a,2a,b,b,b,b) \\
                   &= -a\ln(a)-2a\ln(2a)-4b\ln(b) \\
                   &= -a\ln(a)-2a\ln(2a)-(1-3a)\ln(\frac{1-3a}{4})
        \end{align}

        To find the value of $a$ for which $H(P^*)$ is maximum, we set
        $\frac{dH(P^*)}{da}=0$. 

        We have,

        \begin{align}
            \ln(a) + 2\ln(2a) - 3\ln(\frac{1-3a}{4}) =0
        \end{align}

    \end{solution}

    \part What is the probability that the outcome of $n$ die rolls belongs to
    $P^*$ (and therefore, to $E$ to first order in the exponent)?

    \part (Optional) Verify your answer for n = 250 by Monte Carlo simulation.

\end{parts}
